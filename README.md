# Key-Value Storage on Flash (SSD/MMC Simulator)


## 📌 Описание

Проект представляет собой упрощённую симуляцию работы flash памяти и базового слоя key-value storage.

Реализована модель блочного устройства хранения данных с постраничной записью и поблочным стиранием.


## 🧱 Архитектура проекта

Проект разделён на 3 уровня:

### 1. User layer (demo)
Демонстрационный сценарий, который показывает работу системы:
- запись данных
- чтение данных
- стирание блока
- проверка состояния памяти

---

### 2. Flash simulation layer
Слой эмуляции flash памяти:
- хранение данных в бинарном файле
- постраничная запись и чтение
- поблочное стирание
- использование `0xFF` как состояния “стертая память”

---

### 3. Storage layer
Физическое хранилище:
- файл `flash.bin`
- располагается в директории `storage/`
- создаётся автоматически при запуске

## 🚀 Сборка и запуск

### Сборка
make

### Запуск
./kvs_demo


## 🧪 Демонстрационный сценарий

Программа выполняет последовательность операций:

1. Инициализация flash-памяти
2. Запись данных в страницу
3. Чтение данных из страницы
4. Стирание блока памяти
5. Проверка состояния после erase
